## Quick Start

1. Clone the repository:

```
git clone https://codeberg.org/pharmercy/OverStim.git
cd OverStim
```

2. Set up a virtual environment:

```
python -m venv venv
venv\Scripts\activate
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the program:

```
python OverStim.pyw
```
